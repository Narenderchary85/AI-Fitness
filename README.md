## Installation and Setup

### Backend Repository Setup

1. **Clone the Backend Repository:**
   ```bash
   git clone https://github.com/Narenderchary85/AI-Fitness-B
   
2.  **Install Dependencies:**
     ```bash
      pip install -r requirements.txt

3.  **Environment Variables Setup:**
     ```bash
      FLASK_ENV=development
      MONGO_URI=mongodb://localhost:27017/blogwave
      FLASK_SECRET_KEY=your_super_secret_jwt_key_here
      JWT_SECRET_KEY=key
      PPLX_API_KEY=your_perplexity_api_secret

     
4. **Start the Backend Server:**
     ```bash
          python app.py
          The backend will run on: http://192.168.31.125:5000

### Frontend Repository Setup

1. **Clone the Frontend Repository:**
     ```bash
      git clone https://github.com/Narenderchary85/AI-Fitness-B

2. **Install Dependencies:**
     ```bash
        npm install

3. **Start the Frontend Development Server:**
     ```bash
          npm run dev
          The frontend will run on http://localhost:5173
